## Setup

### Add Jade to Project

1. `File` > `Project Structure`
2. `Libraries` > `+` > `Java`
3. Choose path to `/lib/jade.jar`
4. Click OK and OK again

### Add Jade to Module

1. `File` > `Project Structure`
2. `Modules` > Tab `Dependencies` > `+` > `2. Library`
3. `Project Dependencies` > `jade`
4. Click OK and OK again
